# Contributing

Thanks for taking the time to contribute.

## Repository model

This repository is published as a **public mirror** from a private source repository (using `git subtree`).

That means:

- The public repo is a publish target by default, not the authoritative development workflow.
- Issues are welcome.
- PRs may not be merged (they can be difficult to sync back safely without introducing drift).

## Good ways to help

- File a bug report (include device model, OS version, and reproduction steps).
- Suggest UX improvements (screenshots help a lot).
- Propose test cases and regression checks.
