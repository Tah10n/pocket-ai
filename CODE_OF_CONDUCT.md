# Code of Conduct

This project aims to be welcoming, practical, and respectful for contributors, maintainers, and users.

## Our standards

Examples of behavior that help create a good environment:

- Be respectful in issues, pull requests, reviews, and discussions
- Focus criticism on ideas, code, and product behavior rather than people
- Assume good intent, then clarify disagreements with evidence
- Share context that helps others reproduce, review, and fix problems
- Be thoughtful with security, privacy, and sensitive user data

## Unacceptable behavior

Examples of unacceptable behavior include:

- Harassment, intimidation, threats, or discrimination
- Personal attacks, insults, or abusive language
- Trolling, baiting, or intentionally disruptive behavior
- Publishing private information without explicit permission
- Sexualized language or imagery in project spaces
- Repeatedly derailing technical discussions after moderation

## Scope

This Code of Conduct applies to:

- Issues
- Pull requests
- Code review comments
- Discussions and community conversations tied to this project
- Any other public space used to represent or coordinate project work

## Maintainer responsibilities

Maintainers may take action to protect the project and its contributors, including:

- Editing or removing content
- Locking or redirecting discussions
- Rejecting contributions that do not meet conduct expectations
- Temporarily or permanently restricting participation when necessary

## Reporting

If you experience or witness behavior that violates this Code of Conduct:

- Use private reporting features on GitHub if they are available
- Otherwise contact the repository maintainers through the contact details listed on GitHub
- If a private contact path is not available, open a minimal issue requesting private follow-up without posting sensitive details

When reporting, include links, screenshots, and relevant context if you can do so safely.

## Enforcement principles

Maintainers will review reports in good faith and aim to respond proportionally to the severity and pattern of the behavior. Privacy will be respected as much as practical, but complete confidentiality cannot be guaranteed.
